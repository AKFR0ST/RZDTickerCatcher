# RZDTickerCatcher

Прототип перехватчика выпадающих в продажу железнодорожных билетов. Работает по принципу периодического запроса API rzd.ru и обработки ответа. В дальнейших планах выделить перехватчик в отдельный поток и развернуть многопоточный сервис с полноценным управлением через tg бота(Реализовано в ветке catcherServer). 

## Конфигурирование

Параметры билета, который необходимо поймать, хранятся в конфигурационном файле app.properties

app.urlApi - url API. https://pass.rzd.ru/timetable/public/ru
app.layer_id - layer API, для запросов наличия мест используется значение 5827

app.code0 - семизначный код станции отправления

app.code1 - семизначный код станции прибытия

app.dt0 - дата отправления

app.number - номер поезда

app.type - тип мест [Сид, Плац, Купе, Люкс]

app.maxprice - максимальная цена билета

## Уведомление

В случае появления билета, подходящего под параметры, происходит отправка уведомления в tg бот.
